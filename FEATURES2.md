# Feature Roadmap v2

## High Impact

- [x] **Dictation Mode** (`/dictation`) — Hear a syllable/word → build it on the virtual keyboard.
- [x] **Weak Letter Drill** (`/weak-drill`) — Practice problem characters based on lowest accuracy/easinessFactor.
- [x] **Word Writing Practice** (`/word-writing`) — Draw full Korean words syllable by syllable.
- [x] **Reading Practice** (`/reading`) — Read Korean text, reveal romanization + audio, self-assess.
- [x] **Batchim (Final Consonants)** — CVC syllables (한, 글, 밥). Stage 6 with 24 syllables, 3-key composition in keyboard/dictation/speed-typing.

## Medium Impact

- [x] **Minimal Pair Quiz** (`/minimal-pairs`) — Audio discrimination between confusable character pairs.
- [x] **Speed Typing** (`/speed-typing`) — Timed keyboard composition challenge with WPM tracking.
- [x] **Lesson Cheat Sheet** (`/cheatsheet`) — Reference card for all learned characters by category.
- [x] **Daily Challenge** (`/daily`) — Date-seeded daily puzzle with bonus XP.
- [x] **Progressive Audio Training** (`/listening`) — Listening drill with adjustable speed (0.5x, 0.75x, 1.0x).

## Nice to Have

- [x] **Character Stories** (`/stories`) — Mnemonic + articulation walkthrough per character.
- [ ] **Achievement Showcase** — Visual trophy case (achievements already shown on Dashboard).
- [x] **PWA Offline Support** — Service worker + manifest added.
